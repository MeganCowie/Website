One thing I really want to be able to do with this website is to make figures which people ('users') can interact with. In other words, I want to build applications ("apps", or widgets, or graphical user interfaces) which dynamically update in response to user inputs.


There are a fair few data visualization and analysis platforms for building apps. One example is Montreal's own Plotly Dash. I used Dash during my PhD, and it was a helpful tool for me. Sharing my Dash app with others, however, requires that the app is built by another computer called a server. You can rent servers quite cheaply, but I'm a bit of a "do-it-myself" stickler and found this idea unappealing. The "do-it-myself" avenue would require that I buy a server, but it's expensive, and dealing with hardware upkeep for such a small website doesn't make much sense for me.


PyScript is a brand-new tool that was released in 2022. The appeal is that you don't need a server -- the calculator, controls, callbacks, and plotter can be written right inside the html of this website, and theoretically eliminates my server dilemma.
